# Simple REST

## Installation
### Clone
```
$ git clone https://github.com/tatasfachrul/simple-REST
$ cd simple-REST
$ npm install
```

### Create Environment Variable
```
$ touch .env
$ nano .env
```

```
DB_HOST="Your_Host"
DB_USER="Your_Username"
DB_PASSWORD="Your_Password"
DB_NAME="Your_Table"

SERVER_PORT=9999
```
### Start Development Server
```
$ npm start
```
